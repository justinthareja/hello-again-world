Headings

Headings define the outline of your site. There are six levels of headings.

Lists

This is how you make an unordered list:

  • Add a "ul" element (which stands for unordered list)
  • Add a list item in a "li" tag
  • They don't need to be in any particular order

This is what an ordered list looks like

  1. Notice the new "ol" that wraps the "li" elements
  2. The list items are still the same tag
  3. Also note how the numbers increase all by themselves
  4. You should be noticing things in this order because this is an ordered list

Inline Elements

Sometimes, you need to draw attention to a particular word or phrase. An "em" tag helps here

Other times you need to strongly emphasize the importance of a word. This is when a "strong" tag shines

Other times still, you just need to shout! The combination of and "em" tag and "strong" tag can be quite potent
