Hello, CSS

CSS lets us style HTML elements. There's also another page associated with this example.

List styles

You can style unordered lists with the following bullets:

  • disc
  • circle
  • square

And you can number ordered lists with the following:

  1. decimal
  2. lower-roman
  3. upper-roman
  4. lower-alpha
  5. upper-alpha
  6. (and many more!)

Underlines

Remember meaning should be conveyed through HTML. In order to convey communicate deleted text, the "del" tag can be used. Likewise, any newly added information can be wrapped in a "ins" tag

The Cascade

The lower on the cascade, the higher the priority. Higher priority styles will override any previous ones

  1. The browser's stylesheet
  2. User-defined stylesheet
  3. Exernal stylesheets (that's us)
  4. Page specific styles (also us)
  5. Inline styles (don't do this)
